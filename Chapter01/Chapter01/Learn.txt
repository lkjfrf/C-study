1. cin >> 입력, cout << 출력 (포맷지정 필요없음)
2. C++ 지역변수 선언 어느줄에서 해도 상관없음 C에서는 맨앞줄에 했어야함
3. 함수오버로딩 : 함수이름은 같지만 매개변수의 선언이 다름 -> 하나의 함수이름으로 여러 기능 구현가능
4. 함수 디폴드값은 함수선언 부분에만 써놓아야함
5. 매크로함수는 코드자체를 대체하는 부분이라 속도빠름 / 단점: 정의가 어려움, 한계있음
6. 매크로 한계를 없애줌 -> 인라인 함수 (자료형 정해줘야해서 데이터손실 날수도) -> 템플릿으로 고칠 수 있음
7. using namespace std; 하면 std:: 생략가능
8. 범위지정 연산자 (::) 를 활용해서 전역변수 지역변수 구별가능 -> ::num1 += 8; <- 함수안에 이렇게 하면 전역변수 가르킴
